This project is about creating a open sourced restaurant management system. It is based on NodeJS. We want this to be customized and light weight. No bulky IIS or Apache with things not needed and massive overhead. We are developing in Coffee-Script to make it easier and faster to code. The end product should be a complete system for any size food service store. Whether it is only online or/and order in. It also will be using MongoDB for the data storage. It works well with the javascript based objects that are in this project.
