Welcome to our main site for all our projects. MaadWorks Software is not a company but an organization committed to developing open source software solutions. We provide support for our products but do not charge for the use of our software. You can fork our development and create your own version based on our base work.

We love creating open source software in all kinds of languages. Right now when are using NodeJS and Coffee-Script. We have seen a lot of people against and for these languages. All languages have a place in the coding world. We have design and created projects in ASP.NET with C#, VB.NET. These used MSSQL and javascript to round it out. We also have created Java and C/C++ along with using C# as Mono in the Linux ecosystem. We tend to use the GNU/Linux OS since itself is truly open source and very stable!
